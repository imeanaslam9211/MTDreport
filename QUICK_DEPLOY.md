## ⚠️ Important Notes

### Storage Limitations (Streamlit Cloud Free Tier):
- ❌ Files deleted when app restarts (every few hours)
- ❌ Files deleted after 24h inactivity
- ✅ Use admin panel to re-upload files
- ✅ Consider paid plan for persistent storage

### Security:
- ✅ Keep repo private
- ✅ Change default passwords
- ✅ Don't commit Excel files to GitHub
- ✅ Use HTTPS for all connections

### Performance:
- ✅ App sleeps after 1 hour inactivity
- ✅ First load takes 30-60 seconds
- ✅ Subsequent loads are faster
- ✅ Use "Refresh Data" button to clear cache
